# Global Rules

## Code Execution

- NEVER run code directly (e.g., `python script.py`, `node app.js`, `./binary`)
- NEVER activate virtual environments (e.g., `source venv/bin/activate`, `conda activate`, `. .venv/bin/activate`)
- NEVER execute scripts that could have side effects

You may still use tools like `npm install`, `git`, build commands, and other development utilities - just not running application code or activating environments.
